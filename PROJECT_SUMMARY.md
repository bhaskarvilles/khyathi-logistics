# Khyathi Logistics Solutions - Website Project Summary

## 🎯 Project Overview

A complete, production-ready website for Khyathi Logistics Solutions built with Next.js 16 and shadcn/ui. The website showcases the company's logistics and transportation services with a modern, professional design.

### Technical Stack

- **Framework**: Next.js 16.1.7 (App Router)
- **Language**: TypeScript 5.9.3
- **Styling**: Tailwind CSS 4.2.1
- **UI Components**: shadcn/ui with Radix UI
- **Icons**: Remix Icon (4.9.0)
- **Fonts**: Raleway (sans-serif), Geist Mono
- **Theme**: next-themes for dark mode
- **Notifications**: Sonner for toast messages
