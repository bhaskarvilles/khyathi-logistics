# Khyathi Logistics Solutions Website

A modern, responsive website for Khyathi Logistics Solutions built with Next.js 16 and shadcn/ui.

## 🚀 Features

- **Modern Design**: Clean, professional UI with shadcn/ui components
- **Fully Responsive**: Mobile-first design that works on all devices
- **Dark Mode**: Built-in theme switching capability
- **SEO Optimized**: Proper metadata and semantic HTML
- **Fast Performance**: Built with Next.js App Router for optimal performance

## 📄 Pages

- **Home**: Hero section, services overview, features, and CTA
- **About**: Company information, mission, vision, and technology platform
- **Services**: Detailed service offerings (PTL, FTL, Express Delivery, etc.)
- **Gallery**: Showcase of fleet and operations (placeholder ready)
- **Contact**: Contact form, business information, and emergency support
- **Terms & Conditions**: Legal terms for service usage
- **Privacy Policy**: Data protection and privacy practices
- **Damage & Shortage Policy**: Claims handling procedures

## 🛠️ Tech Stack

- **Framework**: Next.js 16.1.7
- **UI Library**: shadcn/ui with Radix UI primitives
- **Styling**: Tailwind CSS 4.2.1
- **Icons**: Remix Icon
- **Fonts**: Raleway (sans-serif) and Geist Mono
- **TypeScript**: Full type safety

## 🏃 Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```

## 📝 Customization

### Update Company Information

Edit the following files to update company details:
- `components/footer.tsx` - Footer contact information
- `components/header.tsx` - Header navigation and phone number
- `app/contact/page.tsx` - Contact page details
- Policy pages in `app/terms`, `app/privacy`, `app/damage-policy`

### Add Gallery Images

Replace the placeholder content in `app/gallery/page.tsx` with actual images of your fleet and operations.

### Customize Colors

The color scheme can be modified in `app/globals.css` by updating the CSS custom properties in the `:root` and `.dark` selectors.
